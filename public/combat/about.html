About Traveller Combat VTT

üéØ Project Overview

Traveller Combat VTT is a web-based virtual tabletop application for playing Mongoose Traveller 2nd Edition space combat. Built with modern web technologies, it provides a real-time, multiplayer environment for engaging in tactical space battles following the official Traveller rules.

‚ú® Key Features

  • Real-Time Multiplayer: Socket.io-based WebSocket communication for instant battle updates
  • Official Rules Implementation: Faithful implementation of Mongoose Traveller 2E combat mechanics
  • 7 Ship Templates: Scout, Free Trader, Patrol Corvette, Subsidized Merchant, Corsair, System Defense Boat, Close Escort
  • Advanced Weapon Systems: Pulse lasers, beam lasers, missiles with tracking, sandcaster defense, point defense
  • Initiative System: 2D6 + Pilot skill determines turn order each round
  • Hex-Based Movement: Tactical positioning on hex grid with 7 range bands
  • Ship Customization: Modify hull, armor, weapons, and crew for any ship template
  • Solo Mode: Practice against AI opponent with basic tactical decisions
  • Comprehensive Testing: 197 automated tests with 95%+ code coverage
